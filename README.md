# Installation
This configuration is only tested on Windows.

```bash
cd ~
git clone https://github.com/bennyyip/dot-vim.git .vim
cd ~/.vim
```

## For Arch Linux 
```bash
./setup.sh
```

## For other OS
```bash
mkdir .swapfiles
vim +PlugInstall +qall
```

install needed binary and font: 
- ripgrep
- yapf
- inziu-iosevka
